## Transaction Modal

```
+---------------------------------------------------------------------+
| Add New Transaction                                        [X Close] |
+---------------------------------------------------------------------+
|                                                                     |
| Transaction Type:  [Income] [Expense]                               |
|                                                                     |
| Amount: [$_________]                                                |
|                                                                     |
| Description: [_________________________________________________]    |
|                                                                     |
| Category: [___________▼] (Dropdown with categories)                 |
|                                                                     |
| Transaction Date: [__/__/____] (Date picker)                        |
|                                                                     |
| Associate with: (Shows only for expense transactions)               |
| ( ) None                                                            |
| ( ) Goal: [___________▼] (Dropdown with user's goals)               |
| ( ) Envelope: [___________▼] (Dropdown with user's envelopes)       |
|                                                                     |
| Notes: [_________________________________________________]          |
|        [_________________________________________________]          |
|                                                                     |
| [Cancel]                                              [Save]        |
+---------------------------------------------------------------------+
```
